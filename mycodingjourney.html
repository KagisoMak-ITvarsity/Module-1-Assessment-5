The Internet vs The Web

The Internet and the Web are different things:

  • Internet: Network connecting computers (IP)
  • Web: Websites and pages shared over HTTP/HTTPS

My table.html lives on the Web.

Building My Movie Table

I created a simple HTML table showing top-grossing movies:

<table border="1"> <tr> <th>Title</th> <th>Year</th> <th>Box Office</th> </tr> <tr> <td>Avatar</td> <td>2009</td> <td>$2.8B</td> </tr> </table>

How It All Works

When someone visits my page:

  1. Browser sends request: "Hey, give me table.html"
  2. Server finds the file and sends it back with code 200 (success!)
  3. Browser reads the HTML tags like <table>, <tr>, <td>
  4. Browser draws the table on your screen
What I learned: Every time you visit a website, this dance happens. Request → Response → Render. That's it. That's the web.

Learning Web Design

After understanding the basics, I learned how to make websites look better using design principles:

  • CSS for styling: Colors, fonts, spacing, and layouts
  • Visual hierarchy: Making important content stand out
  • White space: Giving content room to breathe
/* Basic CSS styling */ body { font-family: Arial, sans-serif; background: #f5f5f5; padding: 20px; } table { border-collapse: collapse; width: 100%; }

Working with Images

I learned how to add images to my web pages using the <img> tag:

<img src="images/avatar-poster.jpg" alt="Avatar movie poster"> <!-- With width control --> <img src="images/avatar-poster.jpg" alt="Avatar movie poster" width="300">
  • src attribute: Path to the image file
  • File organization: Keeping images in an "images" folder
  • Image formats: JPG for photos, PNG for graphics with transparency

Creating Page Navigation

I learned to connect multiple pages using hyperlinks with the <a> tag:

<!-- Link to About Us page --> <a href="about.html">About Us</a> <!-- Link back to home --> <a href="index.html">Return to Home</a> <!-- Link to contact page --> <a href="contact.html">Contact</a>

This allows users to navigate between different pages:

  • index.html: The home page (main movie table)
  • about.html: Information about the website
  • contact.html: How to get in touch
What I learned: Links are what make the web a "web"—they connect pages together. The href attribute tells the browser where to go when someone clicks.

What I Learned

  • HTML is structure: Tables, rows, cells, images, and links
  • HTTP is communication: Browser talks to server, server talks back
  • Browsers are interpreters: They read my code and make it visual
  • URLs are addresses: They tell browsers exactly where to find files
  • Servers deliver content: My computer became a server when I shared my file
  • CSS is styling: Makes websites look professional and organized
  • Images enhance content: Using proper file paths and alt text
  • Links connect pages: Navigation makes a collection of pages into a website
